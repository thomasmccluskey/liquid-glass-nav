iOS 26 Prototype

Liquid glass navigation

An interactive frosted-glass navigation inspired by Apple's Liquid Glass design language. Click between the labels to feel the pill morph, then dial the sliders below to tune the material in real time.

Material

Each slider drives a single CSS custom property. Defaults match iOS's Regular Material with the new iOS 26 layered specular highlight.

Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step.
